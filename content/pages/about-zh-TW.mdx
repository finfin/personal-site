除了專注於前端開發，也擅長整合數據分析與使用者回饋，以策略性的思維為產品決策提供可靠的基礎。堅信技術的價值不僅在於解決問題，更在於貼近使用者需求，確保每個功能都能帶來實際的影響力。

## **專業技能**

- **前端開發與架構**：多年的前端開發經驗，精通 React ，擅長構建模組化、可維護的代碼結構。具備將大型前端系統進行模組化重構的經驗，並且能靈活運用 JavaScript 及 CSS 進行實作。對前端效能調校也有涉獵，包含 Code Splitting、資源優化、CDN 佈署與更新、快取與載入策略等。
- **專案管理與產品管理**：精通專案管理工具如 ClickUp 與知識管理工具 Notion，管理專案進度、任務分配及團隊協作。具備產品全生命周期管理能力，從需求分析到產品上市的經驗。能夠基於市場和用戶需求快速調整專案優先級，確保產品功能與業務目標一致。
- **數據分析與使用者研究**：結合數據分析與使用者研究，提升產品體驗、優化功能設計。精通使用 Google Analytics、ELK 和 Metabase 等數據分析工具，通過數據洞察來驅動產品決策與優化。同時，擁有進行 UX 使用者研究 的經驗，在產品功能不同階段不同規模下透過不同方法來深入了解使用者需求。
- **提升軟體品質**：推動團隊程式碼的一致性、可讀性與品質。透過團隊的交流如 Code Review，幫助團隊發現潛在的問題、技術債與效能瓶頸，並提升整體軟體的可靠性與可維護性。注重軟體開發流程中的最佳實踐，建立團隊程式碼規範並確保符合團隊需求，進而提升系統的長期穩定性。

## 工作經驗

### **Yourator**, CTO

- **產品策略**：以產品數據分析結合使用者需求推動產品發展。開發 AI 履歷撰寫與 AI 人才篩選/搜尋功能，成功提升履歷完成度 20% 和建立比率 50%。
- **團隊管理與技術提升**：負責產品團隊的招募、培育與績效管理，透過 OKR 驅動團隊成長。導入 ClickUp 和 Notion 改善任務與知識管理，結合 Scrum 提升開發效率。同時透過 Code Review 和技術交流提升團隊技術能力，確保系統穩定與可擴展性。
- **技術決策與系統優化**：根據產品需求與團隊熟悉度，選擇穩定且具效益的技術，例如引入 React 實現前端模組化、ELK 提升搜尋功能準確性。
- **數據驅動的產品優化**：建立 GA、ELK 和 Metabase 的數據分析流程，快速產出產品洞察，支援決策與功能驗證，確保產品持續優化並滿足用戶需求。

### TenMax, 前端技術主管

1. **廣告載入效能優化**： 將廣告載入 script 大小從 30 kb 優化至 16 kb，並實作非同步載入功能，提升廣告載入速度 20%。
2. **多媒體廣告架構設計**： 提出並設計多媒體廣告架構，使用 vanillajs 和 CSS 實現彈性廣告模板載入及多媒體素材播放，演變為公司重要產品。
3. **開發流程優化**：引入 webpack、uglifyjs ，縮減建置時間與大小，建置速度提升五倍。以 Nightwatch 架構 E2E 測試，系統 QA 花費時間縮減一半。

### **Digbil**, 技術主管

1. **數位看板雲端管理系統開發**：主導數位看板管理平台的設計與開發，使用 nw.js 實現廣告展示，解決多螢幕展示技術挑戰。
2. **多螢幕技術應用**：使用 Linux 的 XVFB 管理跨四個螢幕的視窗，提升系統的靈活性與穩定性。
