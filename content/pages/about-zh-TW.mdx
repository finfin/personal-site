## 工作經驗

### **Yourator**, CTO

- **產品策略**：以產品數據分析結合使用者需求推動產品發展。開發 AI 履歷撰寫與 AI 人才篩選/搜尋功能，成功提升履歷完成度 20% 和建立比率 50%。
- **團隊管理與技術提升**：負責產品團隊的招募、培育與績效管理，透過 OKR 驅動團隊成長。導入 ClickUp 和 Notion 改善任務與知識管理，結合 Scrum 提升開發效率。同時透過 Code Review 和技術交流提升團隊技術能力，確保系統穩定與可擴展性。
- **技術決策與系統優化**：根據產品需求與團隊熟悉度，選擇穩定且具效益的技術，例如引入 React 實現前端模組化、ELK 提升搜尋功能準確性。
- **數據驅動的產品優化**：建立 GA、ELK 和 Metabase 的數據分析流程，快速產出產品洞察，支援決策與功能驗證，確保產品持續優化並滿足用戶需求。

### TenMax, 前端技術主管

1. **廣告載入效能優化**： 將廣告載入 script 大小從 30 kb 優化至 16 kb，並實作非同步載入功能，提升廣告載入速度 20%。

2. **多媒體廣告架構設計**： 提出並設計多媒體廣告架構，使用 vanillajs 和 CSS 實現彈性廣告模板載入及多媒體素材播放，演變為公司重要產品。

3. **開發流程優化**：引入 webpack、uglifyjs ，縮減建置時間與大小，建置速度提升五倍。以 Nightwatch 架構 E2E 測試，系統 QA 花費時間縮減一半。

### **Digbil**, 技術主管

1. **數位看板雲端管理系統開發**：主導數位看板管理平台的設計與開發，使用 nw.js 實現廣告展示，解決多螢幕展示技術挑戰。

2. **多螢幕技術應用**：使用 Linux 的 XVFB 管理跨四個螢幕的視窗，提升系統的靈活性與穩定性。
